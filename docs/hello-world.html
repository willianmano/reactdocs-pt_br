Hello World

A maneira mais fácil de começar com React é usando este exemplo de código de Hello no CodePen. Você não precisa instalar nada; você precisa apenas abri-lo em uma nova tab e seguir conosco pelos exemplos. Se você preferir usar um ambiente de desenvolvimento local, veja a página instalação.

O menor exemplo React se parece com isto:

ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('root')
);

É renderizado um cabeçalho com "Hello World" na página;

As próximas seções irão introduzir gradativamente o uso do React para você. Nós iremos examinar os blocks de construção das aplicações React: elementos e componentes. Depois de dominá-los, você criar aplicações complexas com pequenas peças reusáveis.

Uma nota sobre JavaScript #

React é uma biblioteca JavaScript, então assumimos que vocie tem um conhecimento básico da linguagem Javascript. Se vocie não se sente confiante, nós recomendamos refrescar seu conhecimento em JavaScript para então acompanhar mais facilmente.

Nós também usamos alguns exemplos com a sintaxe ES6. Nós tentamos usar com moderação por ser relativamente novo, porém nós encorajamos você a ser familiar com as declarações de: arrow functions, classes, template literals, let, e const.

Você pode user Babel REPL Para verificar o que o código ES6 compila para.
